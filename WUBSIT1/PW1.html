a. When HTML5 has become a standard? What were the main reasons for developing new standard?

a. HTML5 became a W3C recommendation on October 28, 2014. The main reasons for developing this new standard included the need for better support of multimedia and graphics, improved performance and speed, greater consistency in the way web applications behave across different platforms, and an enhanced user experience.

b. What is a difference between W3C HTML5 and WHATWG HTML Living Standard?

b. The W3C HTML5 and WHATWG HTML Living Standard essentially have the same goal, but the difference lies in their development processes. The W3C version is a snapshot of the HTML specification at a particular point in time, while the WHATWG version is continuously updated to reflect ongoing changes and improvements.

d. Comment on semantic and presentational markup. Why presentational markup is considered deprecated now?

d. Semantic markup is based on the meaning of the content, while presentational markup is based on the way content looks. Presentational markup is considered deprecated due to its potential negative impact on accessibility, search engine optimization, and flexibility in the development process. Semantic markup provides better structure and organization for both human readers and machine parsers.
